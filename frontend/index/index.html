🎯 NISA制度とは

NISA(ニーサ)とは、「少額投資非課税制度」の愛称で、投資で得た利益が非課税になる制度です。2024年から制度が大幅に改正され、より使いやすく、投資枠も拡大されました。

🔥 2024年からの主な変更点
  • 非課税保有期間が無期限化
  • 年間投資枠が大幅拡大
  • つみたて投資枠と成長投資枠の併用が可能
  • 生涯非課税限度額が1,800万円に

生涯非課税限度額

新しいNISA制度では、生涯で投資できる非課税枠の上限が設定されています。

📌 生涯非課税限度額の内訳
  • 全体の上限：1,800万円
  • そのうち成長投資枠は：1,200万円まで
  • つみたて投資枠のみの場合：1,800万円まで利用可能
生涯限度額 1,800万円

非課税保有限度額の管理方法

新しいNISA制度では、「簿価残高方式」で非課税保有限度額を管理します。

  • 簿価とは：購入時の金額（取得価額）のこと
  • 売却した場合：売却した分の簿価が非課税保有限度額から減少し、翌年以降に再利用可能
  • 運用益は含まない：株価が上がって評価額が増えても、非課税保有限度額の計算には影響しません
📊 具体例：100万円で購入した株式が150万円に値上がりしても、非課税保有限度額の計算では100万円としてカウントされます。その後売却すると、100万円分の枠が翌年に復活します。

🚀 NISA制度の始め方

対象者

  • 日本に住む18歳以上の方
  • 1人1口座まで開設可能
  • 金融機関は年単位で変更可能

口座開設の流れ

  1. 金融機関を選ぶ
    証券会社や銀行などから選択
  2. 必要書類を準備
    マイナンバーカードまたは通知カード+本人確認書類
  3. 口座開設を申し込む
    オンラインまたは郵送で申込
  4. 税務署の審査
    通常1〜2週間程度
  5. 口座開設完了・取引開始

❓ よくある質問

Q1: 旧NISA口座を持っている場合はどうなりますか？

A: 2023年末までに旧NISA口座で購入した商品は、新しいNISA制度とは別枠で、それぞれの非課税期間が終了するまで非課税で保有できます。2024年からは自動的に新しいNISA口座が開設されます。

Q2: 成長投資枠だけを使うことはできますか？

A: はい、可能です。つみたて投資枠と成長投資枠のどちらか一方だけを使うこともできますし、両方を併用することもできます。

Q3: 非課税期間が無期限とはどういう意味ですか？

A: 購入した商品をいつまでも非課税で保有し続けることができます。旧制度のように「5年後」「20年後」といった期限がありません。

Q4: 海外に転居する場合はどうなりますか？

A: 出国時にNISA口座での新規購入はできなくなりますが、保有している商品は一定期間継続管理勘定で保有できます。帰国後に再開することも可能です。

Q5: 未成年者は利用できますか？

A: 18歳未満の方は通常のNISAは利用できませんが、「ジュニアNISA」制度があります（2023年末で新規投資は終了しましたが、18歳まで非課税で保有可能）。

✨ まとめ

新しいNISA制度の5つのメリット
  1. 非課税期間が無期限 - 長期投資に最適
  2. 年間投資枠が拡大 - 最大360万円まで投資可能
  3. 2つの枠を併用可能 - 柔軟な投資戦略が可能
  4. 売却後の枠が再利用可能 - より自由な運用が可能
  5. 制度が恒久化 - いつでも始められる安心感

新しいNISA制度は、長期的な資産形成を支援するための非常に有利な制度です。投資初心者の方から経験者まで、幅広い方々にとって活用価値の高い制度となっています。
